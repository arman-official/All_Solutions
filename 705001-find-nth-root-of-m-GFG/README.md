# [Find nth root of m](https://www.geeksforgeeks.org/problems/find-nth-root-of-m5843/1)
## Medium
You are given 2 numbers&nbsp;n and m,&nbsp;the task is to find&nbsp;n√m&nbsp;(nth&nbsp;root of m).&nbsp;If the root is not integer then return -1.
Examples :
Input: n = 3, m = 8
Output: 2
Explanation: 23 = 8

Input: n = 3, m = 9
Output: -1
Explanation: 3rd root of 9 is not integer.
Input: n = 4, m = 16
Output: 2
Explanation: 24 = 16
Constraints:1 ≤ n ≤ 90 ≤ m ≤ 20
