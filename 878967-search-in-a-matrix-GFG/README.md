# [Search in a Matrix](https://www.geeksforgeeks.org/problems/search-in-a-matrix--021840/0)
## Easy
Given a 2D integer array mat[][] and&nbsp;a number&nbsp;x, find whether element x is present in the matrix or not.Examples:
Input: mat[][] = [[6, 23, 21],[4, 45, 32],[69, 11, 87]], x = 32
Output: true
Explanation: 32 is present in the matrix, so the output is 1.

Input: mat[][] = [[14, 34, 23, 95, 43, 28]], x = 55Output: false
Explanation: 55 is not present in the matrix, so the output is 0.
Input: mat[][] = [[87, 9, 99],[101, 3, 111]], x = 101Output: true
Explanation: 101 is present in the matrix.
Constraints:1 &lt;= mat.size(), mat[0].size() &lt;= 10001 &lt;= mat[][] &lt;= 1051 &lt;= x &lt;= 105
